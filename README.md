# Loan Approval Automation

## Overview
This project automates the loan approval process using:

- Logistic Regression
- Orange Data Mining
- n8n Automation
- Excel Dataset

## Features
- Loan Approval Prediction
- Automated Workflow
- Email Notifications
- Data Processing

## Tools Used
- Orange
- n8n
- GitHub
- Excel

## Workflow
Customer Data → ML Prediction → Approval/Rejection → Email Notification
